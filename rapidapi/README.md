# EditPDFree PDF API

## Short Description
Professional PDF manipulation API: Convert HTML/URLs to PDF, merge, compress, split, watermark & password-protect documents.

## Long Description

**Transform Your PDF Workflow with EditPDFree PDF API**

EditPDFree PDF API is a comprehensive, production-ready solution for all your PDF processing needs. Built on cutting-edge technology (FastAPI + Playwright), our API delivers high-quality PDF generation and manipulation at lightning speed.

### Key Features

**PDF Generation**
- **HTML to PDF**: Convert HTML content to pixel-perfect PDFs with full CSS support, custom headers/footers, and advanced formatting options
- **URL to PDF**: Capture any webpage as a PDF with complete rendering fidelity
- Customize page size, margins, orientation, scale, and background printing
- Add dynamic headers and footers with HTML support

**PDF Manipulation**
- **Merge PDFs**: Combine multiple PDF files into a single document seamlessly
- **Compress PDFs**: Reduce file size with quality control (low/medium/high compression)
- **Split PDFs**: Extract specific pages or page ranges from any PDF
- **Add Watermarks**: Protect your documents with custom text watermarks (configurable opacity, position, font size)
- **Password Protection**: Secure PDFs with user and owner passwords, restrict printing and copying
